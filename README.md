## Scope

This repository discloses specific non-canonical amino acid (ncAA) modifications to twelve therapeutic peptides, designed to extend their in vivo half-life by conferring resistance to proteolytic degradation. For each peptide, the disclosure includes:

- Native sequence and biological function
- Specific modified sequences with ncAA substitutions
- Rationale for each substitution
- Expected pharmacological impact
- Synthesis considerations

## Non-Canonical Amino Acid Reference

| Abbreviation | Full Name | Mechanism of Action |
|-------------|-----------|-------------------|
| D-Xaa | D-enantiomer of any amino acid | Resists L-specific proteases |
| NMe-Xaa | N-methylated amino acid | Blocks backbone H-bond to protease active site |
| Aib | α-Aminoisobutyric acid | α,α-disubstitution enforces helicity, sterically blocks proteases |
| β-Ala | β-Alanine | Extra methylene in backbone; not recognized by standard proteases |
| Nle | Norleucine | Isosteric Met replacement; eliminates oxidation-susceptible thioether |
| 5-Br-Trp | 5-Bromotryptophan | Halogenation increases metabolic stability and lipophilicity |
| PEG₂-Lys | PEGylated lysine | Increases hydrodynamic radius, reduces renal clearance |
